# Interactive MDX Tutorial

MDX allows you to use components directly in your markdown content! This tutorial demonstrates some of the powerful features you can use.

## JavaScript Expressions

MDX supports JavaScript expressions inline:

- The current year is {new Date().getFullYear()}
- Random number: {Math.floor(Math.random() * 100)}
- String manipulation: {"hello world".toUpperCase()}

## Code Blocks Still Work

```javascript
// Regular code blocks work as expected
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10)); // 55
```

## Best Practices

When using MDX in tutorials:

1. **Keep it Simple** - Don't overuse interactive elements
2. **Accessibility** - Ensure all interactive elements are keyboard accessible
3. **Performance** - Minimize client-side JavaScript
4. **Progressive Enhancement** - Content should work without JavaScript

MDX opens up powerful possibilities for creating engaging, interactive tutorials while maintaining the simplicity of markdown for most content.
